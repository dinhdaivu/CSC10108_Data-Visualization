# Logs Structure

Thư mục này dùng để quản lý bằng chứng AI theo chuẩn vấn đáp.

- `sessions/`: log theo session.
- `prompts/`: prompt thực tế đã dùng.
- `generated/`: output/script từ AI.
- `edited/`: phiên bản do nhóm chỉnh sửa.
- `evidence/`: screenshot, bảng số liệu, output kiểm chứng.
- `archive/`: snapshot cuối kỳ.
