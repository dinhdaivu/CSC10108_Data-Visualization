# Objective 4.2 - Theme 4: Top Commented Products by Year

Use the **Top Commented Products by Year** line chart to identify which Nike products receive the highest number of customer comments in **2024**, **2025**, and **2026**, in order to determine whether review attention is concentrated in a small set of products and how that attention changes over time.

- `Specific`: Product-level comparison of comment volume across years using the most-commented products.
- `Measurable`: Uses `review_count` or `comment_count` by `product_name` from `2024` to `2026`.
- `Achievable`: The required product identifiers, review counts, and review-year fields are available in the prepared review dataset and dashboard visuals.
- `Relevant`: Supports the analytical problem by showing which products generate the strongest customer discussion and how review behavior is distributed across products.
- `Time-bound`: Evaluated across the yearly periods `2024`, `2025`, and `2026`.

## Suggested chart evidence
- Line chart: Axis = `year`, Values = `review_count`, Legend = `product_name` for the top-commented products.
- Optional supporting visual: `Best Rated Categories` for comparison with rating performance.
